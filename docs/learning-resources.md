## 튜토리얼 / 학습 사이트

### WebGL 저수준 학습
- **WebGL2 Fundamentals** ⭐ 강력 추천
  - 최신 WebGL2 기반, 오래된 OpenGL 방식 없음
  - https://webgl2fundamentals.org
  - GitHub: https://github.com/gfxfundamentals/webgl2-fundamentals

- **WebGL2 & GLSL Primer** (2025)
  - 스페이스드 리피티션 방식으로 WebGL2 + GLSL 학습
  - https://github.com/GregStanton/webgl2-glsl-primer

- **The Book of Shaders**
  - GLSL 셰이더 프로그래밍 인터랙티브 학습
  - https://thebookofshaders.com

### Three.js
- **공식 문서:** https://threejs.org/docs/
- **공식 예제:** https://threejs.org/examples/
- **React Three Fiber 문서:** https://r3f.docs.pmnd.rs
- **Drei 문서:** https://github.com/pmndrs/drei

### Babylon.js
- **공식 문서:** https://doc.babylonjs.com
- **Playground (온라인 IDE):** https://playground.babylonjs.com
- **공식 포럼:** https://forum.babylonjs.com

### Phaser (2D)
- **공식 문서:** https://phaser.io/docs
- **공식 예제:** https://phaser.io/examples
- **튜토리얼:** https://phaser.io/tutorials/making-your-first-phaser-3-game

## 추천 학습 순서

```
1. WebGL2 Fundamentals로 저수준 개념 이해
   (삼각형 그리기, 셰이더, 버퍼, 텍스처)

2. 목적에 맞는 라이브러리 선택
   - 3D + React → Three.js + R3F
   - 게임 → Babylon.js 또는 Phaser (2D)

3. 라이브러리 공식 예제/튜토리얼로 실습

4. 실제 프로젝트 제작
```
