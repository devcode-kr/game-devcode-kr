## 요약

웹 기반 게임 개발에서 WebGL은 현재 가장 널리 지원되는 표준이며,
WebGPU는 차세대 API로 주요 라이브러리들이 이미 지원을 추가하고 있다.

### 빠른 선택 가이드

| 상황 | 추천 |
|------|------|
| React 기반 프로젝트 | Three.js + React Three Fiber |
| 게임 엔진 수준의 완성도 필요 | Babylon.js |
| 팀 협업 + 비주얼 에디터 | PlayCanvas |
| 2D 게임 | Phaser or Pixi.js |
| 노코드/교육용 | GDevelop or Construct |
| 완전한 3D 게임 엔진 (WebGL export) | Godot |
